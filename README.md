# Un app para ver perfiles de github

La pagina web [Aquí](https://angelito91.github.io/gh-profile)

![ScreenShot](./public/readme.png)


## 🛠 Instalar
- Debes tener `pnpm` instalado

- Descargar el repositorio
    ```
    git clone
    ```
- Instalar las dependencias
    ```
    pnpm install
    ```
- Ejecuta
    ```
    pnpm run dev
    ```

### 📌 Tecnologías
- Svelte 💊
- Tailwindcss ⭐
- Api Github 💡
